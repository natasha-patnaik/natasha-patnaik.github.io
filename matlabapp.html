MATLAB App

Blue Whale Data Explorer

This app featured a sophisticated user interface that can be shared with other MATLAB users. The primary purpose of this project was to offer a faster, more user-friendly means of interacting with whale data, replacing the traditional method of producing individual reports for various whales and their activities. The app's development included customising the app layout and adding functionality using app properties and callbacks. The app enhanced usability and made the analysis process more engaging and efficient.

The app was designed to create a user-friendly interface where the user can select a Whale ID of their choice from a list provided. On the right-hand side of the screen, the user can switch between tabs to either see some relevant data about the Whale ID they selected (such as their occurence of being spotted throughout the year) or see a geoplot showing the travel path of the selected whale.

The whale data provided in an online course I completed, using which this app was produced.
