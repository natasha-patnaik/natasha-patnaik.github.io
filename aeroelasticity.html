Aeroelastic Simulation

A part of the term two Computing coursework involved producing an aeroelastic simulation of an aerofoil at different velocities over a period of time.

This simulation was produced solely using MATLAB, a programming software, that was also required to save the simulation as an MP4 file as the program was running (producing and saving the video using code, rather than manually). Some parameters were provided as raw data, which then had to be used in calculations (performed using MATLAB code) to generate the coordinates of the aerofoil, as it oscillated in a vertical motion, with the amplitude of each oscillation reducing over time.
